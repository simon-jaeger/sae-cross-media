HTML: Grundlagen

Mit HTML schreiben wir den Inhalt einer Webseite. Ein HTML-Dokument beinhaltet Elemente/Tags, die folgendem Muster folgen:

<tag>Text hier</tag>

Block-Elemente

Erzeugen eine neue Zeile. Folgend einige Beispiele:

  • Paragraph (p)
  • Heading (h1/h2/h3/...)
  • Liste (ul)...
  • ... und verschachtelte Items (li)

Gruppierungs-Elemente

Gruppieren verschachtelte Elemente zusammen. Sind gleichzeitig Block-Elemente. Folgend einige Beispiele:

  • Liste (ul)
  • Komplette Seite (body)
  • Generische Gruppe (div)
  • Semantische Gruppe (header, footer, main, ...)

Inline-Elemente

Fliessen in der aktuellen Zeile. Folgend einige Beispiele:

  • Das ist fett (strong)
  • Das ist kursiv (em)
  • Das ist ein Link (a)

Attribute

Definieren zusätzliche Meta-Informationen, etwa die Ziel-URL eines Links.

<a href="https://www.sae.edu/">SAE</a>

Pfade

Sind entweder relativ, absolut oder extern:

  • Relativ: ../../relativ/to/current/location
  • Absolut: /absolute/from/root
  • Extern: https://eine-andere-website.com

Link zu einer Unterseite.

Medien

Eingebettet über den entsprechenden Tag inkl. Source- und Alt-Attribut:

<img src="./assets/sae-logo.png" alt="SAE-Logo">

Kommentare

Können in den Code eingebettet werden, um ihn zu erklären. Unsichtbar für den Endnutzer.
